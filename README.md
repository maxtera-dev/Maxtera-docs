## 🧩 System Summary

**Backend:** Django REST Framework + PostgreSQL  
**Frontend:** Next.js + TypeScript + TailwindCSS  
**Integrations:** OnePipe API, Data/Airtime Vendors  
**Deployment:** Render / Vercel  
**Versioning:** Git + Semantic Versioning
